Client Work: Q & A

Have any questions that you need answered? Some commonly asked questions are listed on here.
If they're not, you may send an email using this form.

What requests can you do for me?
I can take requests for work done for photography, web design/development (more information below), Adobe Photoshop, Illustrator, InDesign, Flash 2.0/3.0, and Autodesk Maya.
What languages can you code in?
I code in html, css, javascript, php. I am familar with AJAX and mySQL for database.
How long do requests usually take?
Upon recieving a client's request, an estimated time will be sent in a reply back to him/her. The time it would take to complete the request however would vary depending on the type of work requested and the number of other requests waiting.
How will payments be made?
In the reply back to the client, there would be an estimated price. The client will be given a choice to accept that price and pay the first half of the payment before any work will be started. Orders canceled while in progress will not be refunded.
